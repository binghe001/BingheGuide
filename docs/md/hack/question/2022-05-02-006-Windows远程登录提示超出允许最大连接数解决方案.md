# Windows远程登录提示超出允许最大连接数解决方案

当我们拿下一台服务器，用mstsc命令开启远程终端连接服务器时，有时会提示“超出允许最大连接数”，此时，不要慌，这里，我们用以下命令格式：

```bash
mstsc /admin /v:目标ip:端口
```

再次连接，问题解决。
