# Kali运行WPScan报错

报错信息如下。

```bash
Gem load error while running wpscan in kali linux
```

解决方案就是卸载并重装WPScan。

```bash
apt remove wpscan
git clone https://github.com/wpscanteam/wpscan
cd wpscan/
bundle install && rake install
```
