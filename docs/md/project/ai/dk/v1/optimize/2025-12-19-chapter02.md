在2025年9月的云栖大会上，阿里巴巴正式开源了新一代视觉理解模型Qwen3-VL，这一开源举措为多模态AI应用开发带来了新的可能性。随后的10月，阿里又发布了Qwen3-VL-30B-A3B-Instruct版本，该模型在多模态思考、空间感知、智能体能力和开源生态等方面展现出显著优势。

Qwen3-VL的应用场景极为广泛，涵盖图形界面自动化、OCR文档理解、视频内容分析和万物识别等多个领域。今天我将详细介绍如何通过两种不同方式部署Qwen3-VL接口：基于源代码的直接部署和基于Docker的容器化部署

## 一、基于代码部署

### 1.1 下载源码

首先需要获取Qwen3反向代理项目的源代码。这个项目实现了与Qwen3-VL模型的API接口对接。

```bash
git clone https://github.com/binghe001/qwen3-reverse
```

### 1.2 配置环境变量

项目运行需要配置三个关键的环境变量：QWEN_COOKIES、QWEN_AUTH_TOKEN和VALID_TOKENS。首先复制环境变量模板文件：

用下面命令复制.env文件

 ```bash
cp .env.example .env
 ```

**获取 QWEN_COOKIES**：

1. 访问阿里云通义千问的官方聊天界面：https://chat.qwen.ai
2. 完成账号登录后，打开浏览器的开发者工具（F12）
3. 切换到Network（网络）标签页
4. 在聊天界面发送一条任意消息
5. 在网络请求中找到名为`chat/completions`的请求
6. 查看该请求的Headers部分，复制完整的Cookie值

### 1.3 获取 QWEN_AUTH_TOKEN

获取步骤如下：

1. 确保已登录https://chat.qwen.ai
2. 打开开发者工具，切换到Application（应用）标签页
3. 在左侧导航中找到Local Storage（本地存储）
4. 选择[chat.qwen.ai](https://chat.qwen.ai)域名
5. 在右侧的键值对列表中找到"token"项
6. 复制完整的token值，这就是所需的QWEN_AUTH_TOKEN
