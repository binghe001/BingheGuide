## 三、技术选型

高性能Redis组件，在代码结构上非常精简，核心功能就是解决缓存击穿、穿透和雪崩问题。主要的技术选型如下：

* 容器框架：Spring（不强制依赖）
* Redis框架：Redisson（不强制依赖）
* 单元测试：Junit（不强制依赖）
* 基准性能测试：JMH（不强制依赖）
